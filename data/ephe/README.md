# Swiss Ephemeris Data Files

Download the following files from the official Astrodienst repository and place them here:

- `sepl_18.se1` — Planetary ephemeris (outer planets)
- `semo_18.se1` — Moon ephemeris
- `seas_18.se1` — Asteroid ephemeris

Download from: https://www.astro.com/ftp/swisseph/ephe/

Without these files, the system falls back to the built-in Moshier ephemeris (lower precision).
